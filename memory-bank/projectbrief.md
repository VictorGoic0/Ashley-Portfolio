# Project Brief: Ashley Portfolio Website

## Project Overview
A clean, modern portfolio website for Ashley Zheng built with plain HTML, CSS, and JavaScript (no frameworks). The site showcases creative work including images, videos, and PDFs through an elegant card-based interface inspired by modern e-commerce and streaming platforms.

## Core Objectives
1. Display portfolio work in an organized, visually appealing manner
2. Support multiple media types: images, videos, and PDFs
3. Provide smooth navigation between project overview and detail views
4. Maintain clean, professional aesthetic with generous whitespace
5. Ensure full responsiveness across all device sizes

## Key Requirements
- **Technology**: Pure HTML5, CSS3, and vanilla JavaScript (no frameworks or libraries)
- **Design**: Modern card-based layout with 2-column grid (desktop), responsive breakpoints
- **Media Support**: Images, videos, and PDFs with automatic thumbnail generation
- **Navigation**: Main portfolio page with featured section and carousel, plus individual project detail pages
- **Performance**: Fast loading, optimized assets, no external dependencies

## Success Metrics
- Clean, professional appearance
- Smooth navigation and interactions
- Responsive across desktop, tablet, and mobile
- Fast page load times
- Accessible and semantic HTML structure
