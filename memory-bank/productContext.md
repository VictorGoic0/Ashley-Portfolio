# Product Context

## Why This Project Exists
This portfolio website serves as Ashley Zheng's professional showcase, displaying creative work including print ads, lookbooks, video editing projects, and campaign materials. The site needs to present diverse media types (images, videos, PDFs) in an organized, visually compelling way that reflects professional quality.

## Problems It Solves
1. **Media Organization**: Centralizes all portfolio work in one accessible location
2. **Professional Presentation**: Provides a polished, modern interface for showcasing creative work
3. **Multi-Format Support**: Handles different media types (images, videos, PDFs) seamlessly
4. **Responsive Access**: Ensures portfolio is viewable and functional on all devices
5. **Easy Navigation**: Allows visitors to browse all work or focus on featured projects

## How It Should Work
1. **Landing Experience**: Visitors see a prominent header image, portfolio owner information, and featured project cards
2. **Browsing**: Users can scroll through a carousel to see all available projects
3. **Detail View**: Clicking any project card navigates to a dedicated page showing full-resolution media and project description
4. **Visual Flow**: Clean, spacious layout guides the eye naturally through content sections

## User Experience Goals
- **First Impression**: Immediate visual impact with header image and clean design
- **Discoverability**: Easy to find and browse all available work
- **Detail Exploration**: Seamless transition to detailed project views
- **Professional Feel**: Polished, modern aesthetic that reflects quality of work
- **Accessibility**: Works well on any device, any screen size
